## What We're Building

Tetris — falling pieces, rotation, line clearing, and increasing drop speed. The finale of the eight-game series.

The coding challenge: Tetris needs a 2D grid that remembers where every settled block is. A falling piece is just a shape on top of that grid — when it lands, it merges in. When a full row of data exists, delete it and shift everything down. That's the whole game, built from one data structure.

Understanding this makes every grid-based game readable — Minesweeper, Sudoku, Connect Four, any game on a board.
