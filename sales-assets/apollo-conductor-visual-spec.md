# Apollo Conductor — Visual Spec
## Architecture Diagram: Image Generation Prompts

This document contains ready-to-use Gemini/Imagen prompts for generating the Apollo-as-conductor architecture visuals. CMO uses these when creating social assets, pitch decks, or web graphics for Athena and Pantheon.

---

## Visual Concept

**The story in one image:**  
A person at the top sends a message. Apollo — glowing at the centre — receives it and fans out instructions to a team of specialist agents below. Results flow back up through Apollo to the person.

**This replaces** any "stack of tools" or "dashboard" visual. Apollo is always the hub. The client is always singular, at the top. The team is always beneath Apollo, not beside it.

---

## Prompt 1: Athena Architecture Diagram (Square 1:1)

Use for: LinkedIn posts, social media, pitch deck slides

```
Create a bold, modern tech graphic in 1:1 aspect ratio. Dark black background (#000000 to #1A0F00 gradient). 

Layout (top to bottom):
- TOP: A single glowing chat bubble icon (white/light, soft glow) labelled 'YOU' in clean white text
- CENTRE: A large, prominent glowing orange brain/node icon labelled 'APOLLO' in bright orange (#FF6600) with amber halo (#FFB347). This is the centrepiece — make it visually dominant
- BOTTOM: Three smaller glowing icons in a row, connected to Apollo by converging lines in orange: labelled 'CMO', 'SCOUT', 'OPS'
- Connecting lines flow from YOU down to APOLLO, then fan out from APOLLO down to the three agents
- The visual hierarchy is clear: one client, one conductor, three specialists

Style: Clean, circuit-board aesthetic. Subtle gold particle effects. No people, no photos. Orange (#FF6600) as primary accent. White text labels in a modern sans-serif font.

Below the diagram, add a tagline in bright orange: "You talk to Apollo. Apollo runs the team."
```

## Prompt 2: Pantheon Architecture Diagram (Wide 16:9)

Use for: pitch deck hero slide, website header, presentation

```
Create a bold, modern tech graphic in 16:9 widescreen format. Dark black background (#000000 to #1A0F00 gradient).

Layout (top to bottom, centred):
- TOP: A single glowing chat bubble / person icon (white, soft glow) labelled 'YOU' in white
- CENTRE: Large, glowing orange node labelled 'APOLLO' in bold orange (#FF6600). Amber outer glow ring (#FFB347). This is clearly the most visually prominent element on the canvas — the conductor
- BOTTOM: Five smaller glowing orange node icons in a horizontal row, each connected to APOLLO by fanning orange circuit lines. Labels in white: 'CTO', 'CMO', 'CFO', 'SCOUT', 'ENGINEER'
- All connections animate-style (as if electricity is flowing from Apollo outward)
- In the bottom-right corner, a small block: black background, orange text: 'Your dashboard. Your data. Your team.'

Style: Cinematic tech aesthetic. Circuit board patterns in the dark background areas. Gold/amber particle effects concentrated around APOLLO. No people, no photos. Clean, modern sans-serif throughout.

Top-left corner: Cognito Coding logo (small, white/orange).
```

## Prompt 3: Product Family Tier Visual (Square 1:1)

Use for: social posts explaining the Apollo → Athena → Pantheon journey

```
Create a bold, modern infographic in 1:1 aspect ratio. Dark black background.

Show three tiers stacked vertically, each with increasing size/scale to show progression:

TIER 1 (top, smallest): Orange pill label 'APOLLO £29.99/mo' — small glowing brain icon — tagline 'Start with knowledge.'

TIER 2 (middle, medium): Orange pill label 'ATHENA £750/mo' — medium glowing brain icon with two smaller icons beside it — tagline 'Add wisdom.'

TIER 3 (bottom, largest, most prominent): Orange pill label 'PANTHEON £2,500+' — large glowing brain icon with five smaller icons fanning out below — tagline 'Build your temple.'

An orange arrow connects TIER 1 → TIER 2 → TIER 3 along the left side, suggesting an upgrade journey.

Background: Dark gradient with subtle circuit board patterns. Gold particle effects increase in density toward the bottom (more power = more sparkle).

Style: Clean, modern, bold. No clutter. This is a ladder to climb, not a menu to browse.
```

## Prompt 4: Apollo Hero Visual (Square 1:1)

Use for: Apollo product page, Apollo outreach emails, social posts about Apollo specifically

```
Create a bold, modern social media tech graphic 1:1 aspect ratio, dark background, bright orange accents.

Central image: A glowing orange brain icon, floating slightly above centre, with a soft amber halo. Around it in orbit: small icons representing email (envelope), calendar (grid), research (magnifying glass), tasks (checkmark), all connected by thin orange circuit lines to the brain.

Above the brain: a faint chat bubble (white, transparent) containing three dots — suggesting conversation / natural language.

The overall effect: the brain is alive, connected, always listening. It's a personal intelligence at the heart of your business.

Bottom text in clean white: 'APOLLO' — and in smaller orange text below: 'Your AI Executive Assistant'

Background: #000000 to #1A0F00 dark gradient with subtle gold particle effects.
```

## Usage Notes for CMO

1. For any Athena or Pantheon post, **always use Prompt 1 or Prompt 2** (not a generic tech image) — the architecture diagram is the core message
2. Apollo must always be visually dominant (larger, brighter, more central than any specialist agent)
3. The client (YOU) must always appear above Apollo — never beside it, never below it
4. Specialist agents must always appear below Apollo — never beside it at the same level as the client
5. Taglines must always appear: "You talk to Apollo. Apollo runs the team." (Athena) or "You never manage the stack — you manage Apollo." (Pantheon)
