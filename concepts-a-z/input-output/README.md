## The Walkie-Talkie Trick

Ever used a walkie-talkie? You press the button, say something, then let go and wait. A moment later, a voice comes back. That's exactly how a program talks to you. You type something in — that's the program **listening** (input). Then it says something back — that's the program **answering** (output).

Two lines of Python do the whole thing.

---

## The Code

```python
name = input("What's your name? ")
print("Hello " + name + "!")
```

The first line asks a question and waits for you to type your name — it stores your answer in a box labelled `name`. The second line takes what's in that box, adds `"Hello "` in front, and shows it on the screen.

You talked. It answered. That's input and output.
