## The Analogy

A light switch has two settings — on or off. Nothing in between. No "sort of on."

Computers work exactly the same way. A **Boolean** is a value that can only ever be `True` or `False`. No maybe. No "it depends."

---

## What It Does

```python
light_on = True
door_locked = False

if light_on:
    print("The light is on!")
```

When Python checks `light_on`, the answer is just `True`. That one simple idea — yes or no, true or false — powers every `if` statement, every loop condition, every game logic check your code will ever run.

---

## `True` or `False` — That's It

`bool` is the type name. `True` and `False` are its only two values. Capitalise them — Python is case-sensitive and `true` without the capital throws a `NameError`.
