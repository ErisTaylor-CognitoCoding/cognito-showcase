## What the AI Built

A Python app that monitors Creatures of Sonaria gameplay and auto-saves the best moments. When a kill-confirm trigger fires, the app grabs the clip with surrounding context and surfaces it in a sleek review interface. Drag to keep, swipe to discard. Everything runs inside Replit — no desktop software needed.
