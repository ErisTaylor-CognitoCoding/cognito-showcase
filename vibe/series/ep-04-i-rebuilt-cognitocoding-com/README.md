## The Prompt I Gave

Not a design spec — just one pasted company overview. Who Cognito Coding is now, where it's heading, the whole new direction, in plain English. The agent stopped and planned first: read the brief, reviewed the existing site, and turned that paragraph of intent into an ordered task list before touching a single file.

## What the AI Built

A repositioned website, not a redesign. Every corner of cognitocoding.com now says the same thing: *Learning To Code By Solving Real Problems.* 

Changes in the build:
- Navbar tabs with a custom smooth-scroll hook (the normal hash-link approach gets killed by scroll-to-top)
- Footer flipped from WhatsApp to YouTube; added GitHub showcase link
- Tagline: "Learn To Code By Building Real Things" → "Learning To Code By Solving Real Problems" with gradient on the punchy half
- Apps grid cut to the four that are actually live: Chatbots, WagTracker, Solo Tutor Lite, Award Tracker
- About page rewritten first-person — genuine bio kept (25+ years teaching, coding since the 80s, Replit Ambassador), frame changed from selling to sharing
- Caught a sneaky white-on-white contrast bug after the theme flip — always re-check contrast after a palette change
- Nova chatbot voice updated so it doesn't contradict the new homepage — if the bot contradicts the site, the pivot leaks
- Privacy policy treated as a legal document: confirmed processing relationships before trimming, updated data-controller wording, bumped "last updated" date

**Honest verdict:** 45 minutes end-to-end. The agent planned before building — that's the skill that makes vibe coding produce something coherent rather than a mess.
