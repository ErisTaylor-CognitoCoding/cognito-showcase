## The build

No scripted demo. Built live on camera, including the debugging.

The goal: a chatbot that sits on the Cognito Coding website, answers prospect questions 24/7, and captures name + email when someone's genuinely interested. Wired to Claude via Replit's Anthropic Connector — no hardcoded API keys (saying this out loud is the #1 beginner mistake to avoid).

**Key steps on camera:**

1. Spin up the chat UI on Replit
2. Connect to Claude using Replit's Anthropic Connector
3. Seed the knowledge base from cognitocoding.com pages
4. Write the system prompt — on-brand answers + lead capture behaviour
5. Store conversations so no lead slips through
6. Debug it live when it broke

---

## What was built

A lead-capture chatbot that:

- Answers prospect questions about Cognito Coding
- Stays on-brand (seeded from the live website)
- Asks for name + email when someone's genuinely interested
- Stores every conversation for follow-up

Stack: Replit · Claude (via Anthropic Connector) · cognitocoding.com
